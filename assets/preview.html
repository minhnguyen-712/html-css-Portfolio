How to use

  1. Put your PDF(s) and image(s) in assets/ or assets/media/.
  2. In VS Code, click a PDF file in Explorer — with the workspace setting added, it should open using the PDF preview extension instead of showing raw bytes.
  3. To view the HTML preview page: open assets/preview.html in a browser or use the Live Server extension (right-click → Open with Live Server).
